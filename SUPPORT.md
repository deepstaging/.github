### Issues

For bugs and feature requests:

- [Deepstaging Issues](https://github.com/deepstaging/deepstaging/issues)
- [Roslyn Toolkit Issues](https://github.com/deepstaging/roslyn/issues)

## Before Asking

1. Check the documentation
2. Search existing issues and discussions
3. Try to create a minimal reproduction

## Reporting Bugs

When reporting bugs, include:

- Package version
- .NET version
- Minimal code to reproduce
- Expected vs actual behavior
- Error messages or stack traces

## Feature Requests

When requesting features:

- Describe the problem you're trying to solve
- Explain your current workaround (if any)
- Provide examples of how you'd use the feature
