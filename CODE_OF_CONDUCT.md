# Code of Conduct

## Our Standards

We're committed to providing a welcoming environment for everyone. We expect:

**Positive behaviors:**
- Being respectful and inclusive
- Giving and receiving constructive feedback gracefully
- Focusing on what's best for the community
- Showing empathy toward others

**Unacceptable behaviors:**
- Harassment, trolling, or personal attacks
- Publishing others' private information
- Discriminatory language or imagery
- Other conduct inappropriate in a professional setting

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that don't align with this Code of Conduct.

Instances of abusive behavior may be reported by contacting the project team. All complaints will be reviewed and investigated.

## Scope

This applies to all project spaces and when representing the project publicly.
